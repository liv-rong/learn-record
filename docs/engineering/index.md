# 前端工程化

## 什么是前端工程化

- 标准化流程 （代码风格）
- 开发流程优化（构建、编译、打包）
- 代码规范与质量（ESLint、Prettier、TypeScript）
- 模块化与组件化（组件库、Monorepo）
- 自动化（CI/CD、测试、部署 Docker GitHub Actions）
- 性能优化（Bundle 分析、懒加载、缓存）
